# XtoP - Smart Excel to PowerPoint Report Generator. 

## Overview

XtoP automatically converts Excel business data into professional PowerPoint reports.

## Features

- Excel Upload
- KPI Dashboard
- Revenue Trend Charts
- Regional Performance Analysis
- Product Revenue Analysis
- Employee Performance Analysis
- Auto Generated Insights
- PowerPoint Report Generation
- Streamlit Frontend

## Tech Stack

- Python
- Pandas
- Matplotlib
- python-pptx
- Streamlit

## Workflow

Excel File
↓
Data Analysis
↓
Chart Generation
↓
Insight Generation
↓
PowerPoint Creation
↓
Download PPT

## Run Locally

pip install -r requirements.txt

streamlit run app.py
